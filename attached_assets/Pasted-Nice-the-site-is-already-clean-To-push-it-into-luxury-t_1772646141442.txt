Nice — the site is already clean. To push it into “luxury” territory, you need fewer “app-startup” cues (pastels, bright orange gradients) and more high-end editorial + glass + subtle lighting.

Below are surgical changes you can apply today (mostly CSS token + a few class swaps in Home.tsx). No redesign, just luxury uplift.

1) Make gold feel “premium” (less orange, more champagne)

Right now your palette leans a bit orange (especially --secondary and some gradients). Luxury gold should read champagne.

Update in client/src/index.css (tokens only)

Replace these:

--primary: 43 65% 55%;
--secondary: 30 90% 55%;
--accent: 30 90% 55%;

With:

/* Champagne gold + restrained accent */
--primary: 43 55% 58%;         /* softer gold */
--secondary: 32 35% 46%;       /* muted bronze (not orange) */
--accent: 43 55% 58%;          /* keep accent aligned to gold */
--muted: 220 28% 18%;          /* deeper navy-muted */
--muted-foreground: 45 18% 74%;

Why this matters: the site stops screaming “orange”, and starts reading “luxury navy + champagne”.

2) Add a subtle “luxury noise + vignette” overlay (instant premium)

This is the easiest high-impact upgrade.

Add to index.css

  overflow: hidden;

Your glass-card class exists but your key sections still use flat bg-card border.

Apply it to your featured book sections

In Home.tsx, change wrappers like:

<div className="bg-card border rounded-[2.5rem] ...">

To:

<div className="lux-card rounded-[2.5rem] ...">

Result: premium depth and consistent glass aesthetic.

4) Remove the pastel image panels (they read “template”)

Your app image containers use bg-rose-50, bg-teal-50, bg-indigo-50. That’s not luxury.

Replace each app image panel background

Change:

<div className="... bg-rose-50 ...">

To:

<div className="... bg-white/5 ...">

Same for teal/indigo. Keep the small accent dot if you want category color—but don’t let it dominate the card.

Result: cohesive luxury palette across the entire page.

5) Make typography more “editorial”

You already use Inter + Plus Jakarta Sans. Luxury feel improves when:

body text is slightly smaller + higher line-height

headings slightly tighter + less “orange gradient”

In Hero headline gradient

Current:

bg-gradient-to-r from-primary via-orange-400 to-primary

Replace with:

bg-gradient-to-r from-primary via-[#ffe3a6] to-primary

It becomes “gold shimmer”, not orange neon.

6) Micro-interactions: slower, softer, less bouncy

You have hover lifts and strong transforms. Luxury sites use subtle movement.

For app cards, reduce hover translation:

hover:-translate-y-2

→

hover:-translate-y-1

And reduce image scale:

group-hover:scale-105

→

group-hover:scale-[1.02]
