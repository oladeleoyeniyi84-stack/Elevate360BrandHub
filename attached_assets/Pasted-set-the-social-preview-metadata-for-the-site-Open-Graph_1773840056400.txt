set the social preview metadata for the site:

Open Graph tags for IG/FB/LinkedIn/WhatsApp and many others

Twitter/X Card tags for X

a proper preview image URL

This only affects how the site looks when shared, not how the website itself renders.

What to add in the <head>

Use this in your main HTML head, or in the SEO/meta component if your Replit app is React/Vite/Next.

<meta property="og:type" content="website" />
<meta property="og:site_name" content="Elevate360Official" />
<meta property="og:title" content="Elevate360Official | Empowering Lives Through Technology & Words" />
<meta property="og:description" content="Explore Elevate360Official — a digital ecosystem of apps, books, music, art, and creativity designed to elevate everyday life." />
<meta property="og:url" content="https://www.elevate360official.com/" />
<meta property="og:image" content="https://www.elevate360official.com/social-preview/elevate360-logo-share.png" />
<meta property="og:image:secure_url" content="https://www.elevate360official.com/social-preview/elevate360-logo-share.png" />
<meta property="og:image:width" content="1200" />
<meta property="og:image:height" content="630" />
<meta property="og:image:alt" content="Elevate360Official logo" />

<meta name="twitter:card" content="summary_large_image" />
<meta name="twitter:title" content="Elevate360Official | Empowering Lives Through Technology & Words" />
<meta name="twitter:description" content="Explore apps, books, music, art, and creativity at Elevate360Official." />
<meta name="twitter:image" content="https://www.elevate360official.com/social-preview/elevate360-logo-share.png" />
Best image path

Upload your generated logo share image to something like:

/public/social-preview/elevate360-logo-share.png

or, if your app uses client public assets:

/client/public/social-preview/elevate360-logo-share.png

The key is that this file must be publicly reachable at:

https://www.elevate360official.com/social-preview/elevate360-logo-share.png
Important

Do not use a tiny logo file alone for social sharing.
For best results, create a 1200 × 630 branded share image with:

dark navy background

official gold rocket logo

Elevate360Official brand name

maybe the tagline: Empowering Lives Through Technology & Words

That will preview much better across platforms.

After deploying

Most platforms cache previews, so after deployment you may need to refresh them using:

Facebook Sharing Debugger

LinkedIn Post Inspector

X Card Validator if available

WhatsApp usually refreshes after cache clears or after changing the URL query once
